Download this folder and save it wherever you save your other macros (or use git clone).

Add custom macro button:

Step 1: Open an assembly, right click on top toolbar and select the "Customize" option.

![Step 1 open the customize dialog.](/HowTo/step1.png)

Step 2: Click Commands > Macro > and drag the “New Macro Button” onto the toolbar.

![Step 2 add macro button.](/HowTo/step2.png)

Step 3: Locate the macro file and set the image and the description to use.

![Step 3 set macro details](/HowTo/step3.png)

(if you want to make your own icon, be sure to save it in a BMP in 24-bit format (RGB) with the option to write color space information turned off).
